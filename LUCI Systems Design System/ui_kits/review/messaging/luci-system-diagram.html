FLOORMAP ICON OPTIONS

Two AutoCAD-style variants per element. Open the icon preview page for large renders on disk-colored viewports — then reply with letter picks (e.g. Audio A, Screen B, Rack A…).

  • AAudio Zone

    A — dashed ellipse, speaker node, wave arcs. B — dashed rect, diagonal hatch, corner speakers.

  • BScreen endpoint

    A — monitor + X overlay + stand tick. B — crosshair circle + mount block (plan view).

  • CRack / Head end

    A — EIA rack elevation, shelf lines, status lamps. B — cabinet outline, U-slot blocks.

  • DVendor

    A — diamond block ref, leader line, label box. B — equipment flag tag with pin.

  • EComputer

    A — monitor plan, keyboard line, desk tick. B — tower + monitor pair.

  • FVideo Wall

    A — tile grid, mullion lines. B — wide plane, diagonal brace, vertical seams.
