The Embedded Operation orbit is the canonical diagram for LUCI as one accountable team wrapping six services around the client. Originally built for page 4 of the capabilities document, it now lives here as a standalone, reusable asset — drop it into any deck, doc, email, or web page without rebuilding.

FILES

  • SVGVector master (editable, scalable)

    assets/diagrams/embedded-operation-orbit.svg — transparent background, live text in Space Grotesk. Preferred for web, slides, and print. Edit copy or geometry directly in the file.

  • PNGRaster export (3× / 2700×1440)

    assets/diagrams/embedded-operation-orbit.png — transparent background. Use where SVG isn't supported (some email clients, raster pipelines).

HOW TO PULL IT IN

Reference the SVG by relative path — no rebuild needed. From a sales asset in ui_kits/sales/:

<img src="../../assets/diagrams/embedded-operation-orbit.svg"
     alt="The Embedded Operation"
     width="680">

The aspect ratio is 900 × 480 (15:8). Set width and let height scale; the SVG is resolution-independent. On dark backgrounds, use the navy/light variant instead (the labels here are near-black for light canvases per Track C).
