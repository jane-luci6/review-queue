USAGE RULES
Patterns — Use as background of a single feature section, not the entire page. Pair colors carefully: dark patterns on dark bg, light patterns on light bg. Apply background-size: cover or 800px for tiling. Keep at full opacity — they're already subtle.
Textures — Apply as a full-bleed overlay position: absolute; inset: 0; mix-blend-mode: color-burn; opacity: 0.5–0.94. Stack over a flat-color base. Never use as a primary background.
One per page — A single hero with texture and a single section with a pattern is the maximum. Multiple decorative surfaces compete for attention and dilute the brand's clean, technical voice.
Never combine — Don't stack a pattern and a texture in the same section. Don't tile patterns at a small scale (looks busy). Don't tint patterns to a different color than provided.
